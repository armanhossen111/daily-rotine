🧍 Body Care

  • Standing stretches done
  • Sitting stretches done

🌇 Post-Shift Transition

  • Changed clothes

  • No phone (20 min)

🎮 Dopamine Control

  • 1 activity only
  • Timer set (1 hour)
  • Stopped by 9:30 PM

🌙 Sleep Wind-Down

  • Blue light filter ON
  • Stretching done
  • Phone away from bed
